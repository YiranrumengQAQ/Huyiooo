# 智能水印大师 (WATERMARK PRO 2025)

一款基于纯前端技术开发的在线智能水印添加工具，支持图片上传、自定义水印内容和样式调整，无需后端服务器即可完成水印添加和图片导出。

## 功能特性

### 图片管理
- 支持多种格式：JPG、PNG、WEBP、GIF
- 拖拽上传或点击选择图片
- 实时预览效果

### 水印编辑
- 自定义文字水印内容
- 一键插入当前时间
- 一键插入日期信息
- 可视化拖拽定位

### 样式定制
- 字体大小调节
- 透明度控制（0-100%）
- 旋转角度调整（0-360°）
- 自定义文字颜色
- 粗体样式切换
- 阴影效果开关

### 位置控制
- 精确坐标定位
- 点击水印框拖动移动
- 实时位置显示

### 导出功能
- 一键下载带水印图片
- 重置所有设置
- 保持原图质量

## 技术栈

- **HTML5**：构建页面结构
- **CSS3**：现代化UI设计和动画效果
- **JavaScript**：核心功能逻辑实现
- **Canvas API**：图像处理和水印绘制

## 项目亮点

- 纯前端实现，无需服务器，保护用户隐私
- 响应式设计，支持各种屏幕尺寸
- 直观的用户界面，操作简单易上手
- 实时预览，所见即所得
- 单文件部署，方便快速使用

## 使用方法

1. 打开 HTML 文件或部署到 Web 服务器
2. 点击或拖拽上传需要添加水印的图片
3. 在水印内容区域输入文字或插入时间/日期
4. 调整水印样式（大小、颜色、透明度等）
5. 拖动水印框调整位置或使用精确坐标
6. 点击"下载图片"保存带水印的图片

## 适用场景

- 个人照片版权保护
- 社交媒体图片标注
- 产品图片品牌标识
- 文档图片时间戳记录
- 批量图片快速处理

## 部署说明

本项目为单 HTML 文件应用，可通过以下方式使用：
- 直接在浏览器中打开 HTML 文件
- 部署到任意静态网站托管服务（GitHub Pages、Netlify 等）
- 放置在本地或远程 Web 服务器

无需安装依赖，开箱即用。
