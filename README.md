# 🎨 智能水印大师 (WATERMARK PRO 2025)

一款功能强大的纯前端在线智能水印添加工具，提供丰富的字体样式定制选项和可视化编辑界面，让水印添加变得简单而专业。

**在线体验**: [https://yiranrumengqaq.github.io/Huyiooo/](https://yiranrumengqaq.github.io/Huyiooo/)

## ✨ 核心功能

### 📷 图片管理
- **多格式支持**：JPG、PNG、WEBP、GIF 全格式兼容
- **灵活上传方式**：支持点击选择或拖拽上传
- **实时预览**：上传即显，所见即所得

### ✏️ 水印内容编辑
- **自定义文字**：支持任意文本内容输入
- **动态时间插入**：一键插入当前时间，自动更新
- **日期快捷添加**：快速添加日期信息
- **多行文本支持**：可添加复杂的多行水印内容

### 🎨 丰富的字体样式定制

#### 字体外观
- **字体大小调节**：通过滑动条或数字输入精确控制字体大小（支持从小号到超大号）
- **HTML5 颜色选择器**：提供标准颜色选择面板，支持：
  - 预设颜色快速选择
  - 自定义 RGB 颜色
  - 十六进制颜色代码输入
  - 实时颜色预览
- **粗体效果**：一键切换粗体样式，让水印更醒目
- **字体粗细控制**：支持不同字重的文字显示

#### 视觉效果
- **透明度控制**：
  - 范围：0-100% 精确调节
  - 滑动条实时调整
  - 百分比数值显示
  - 适配不同背景，保持美观
  
- **旋转角度调整**：
  - 支持 0-360° 任意角度旋转
  - 数字输入或滑动条操作
  - 实时预览旋转效果
  - 适用于斜向水印、防盗用标记
  
- **阴影效果**：
  - 一键开启/关闭文字阴影
  - 增强水印可读性
  - 自动适配文字颜色
  - 让水印在复杂背景下更清晰

### 📍 精确位置控制
- **可视化拖拽**：点击水印预览框直接拖动移动位置
- **精确坐标定位**：通过数字输入设置 X、Y 坐标
- **实时坐标显示**：拖动时同步更新坐标数值
- **像素级精度**：满足专业排版需求

### 💾 导出与管理
- **高质量下载**：保持原图分辨率和质量导出
- **一键重置**：快速清空所有设置，重新开始
- **无损处理**：不压缩原图，保证输出品质

## 🎯 字体样式功能详解

### 颜色系统
本工具采用 HTML5 标准颜色选择器，提供专业的颜色选择体验：

1. **预设颜色板**：常用颜色快速选择
2. **自定义调色板**：RGB 三通道精确调节
3. **颜色历史记录**：浏览器自动保存最近使用的颜色
4. **十六进制支持**：可直接输入 HEX 颜色代码（如 #FF5733）
5. **实时预览**：选择颜色即时反映在水印上

### 透明度应用场景
- **10-30%**：极淡水印，适合需要低调标识的场景
- **40-60%**：半透明效果，平衡美观与版权保护
- **70-90%**：标准水印，清晰可见但不过分突出
- **100%**：完全不透明，强版权保护或重要标注

### 旋转角度建议
- **0°**：标准水平水印
- **45°/-45°**：经典斜向水印，常用于防盗图
- **90°/-90°**：垂直水印，适合侧边标识
- **自定义角度**：满足各种创意设计需求

### 阴影效果优势
- 在浅色背景上使用深色文字时，添加阴影增强对比度
- 在深色背景上使用浅色文字时，阴影提升层次感
- 复杂图案背景下，阴影让文字更易识别
- 专业级视觉效果，提升水印质感

## 📖 使用指南

### 快速开始
1. 访问在线地址：[yiranrumengqaq.github.io/Huyiooo/](https://yiranrumengqaq.github.io/Huyiooo/)
2. 点击或拖拽上传需要添加水印的图片
3. 在「水印内容」输入框输入文字，或使用快捷按钮插入时间/日期
4. 使用右侧样式面板调整字体效果：
   - 拖动「字体大小」滑块调整大小
   - 点击「文字颜色」按钮选择颜色
   - 勾选「粗体」和「阴影效果」增强视觉
   - 调整「透明度」和「旋转角度」滑块
5. 点击水印预览框拖动调整位置，或在「精确坐标」输入数值
6. 点击「💾 下载图片」保存带水印的图片

### 高级技巧
- **批量处理**：下载后重新上传新图片，水印设置自动保留
- **精确定位**：使用坐标输入可实现像素级精确摆放
- **颜色搭配**：使用取色器选择与图片协调的水印颜色
- **透明度叠加**：低透明度+阴影效果，打造若隐若现的高级感
